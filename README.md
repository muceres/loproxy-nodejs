# loproxy-nodejs

> Request the target of your choice by passing through one of your local port as a proxy


## Install

```
git clone https://github.com/muceres/loproxy-nodejs
cd loproxy-nodejs
npm install
```


## Usage

```bash
PORT=3000 BASE_URL=https://google.com node index.js
```
